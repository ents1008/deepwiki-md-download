# deepwiki-md-download

DeepWiki 页面导出 Markdown 的油猴脚本。

源仓库（原始 Chrome 扩展）：`https://github.com/zxmfke/deepwiki-md-chrome-extension`

## 安装（正确方式）

前置：先安装油猴扩展（Tampermonkey 或 Violentmonkey）。

### 方式 1：点击安装链接

- GitHub 安装链接：`https://github.com/ents1008/deepwiki-md-download/raw/main/deepwiki-md.user.js`
- Raw 安装链接：`https://raw.githubusercontent.com/ents1008/deepwiki-md-download/main/deepwiki-md.user.js`

打开链接后，油猴会弹出“安装脚本”页面，点“安装”。

### 方式 2：本地导入 `.user.js`

1. 下载仓库里的 `deepwiki-md.user.js`
2. 打开油猴管理面板
3. 使用“导入/Install from file”导入该 `.user.js` 文件

## 使用

1. 打开 `https://deepwiki.com/*` 页面
2. 右下角面板功能：
   - 导出当前页 Markdown
   - 批量导出全部页面 ZIP
